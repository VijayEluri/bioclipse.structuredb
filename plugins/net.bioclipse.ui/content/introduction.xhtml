Welcome to Bioclipse



Introduction

Bioclipse is an open source workbench for life science. This means, you can use it to import/export/edit/visualize/convert etc entities in chemistry (chemoinformatics), biology and biotechnology (bioinformatics), medicine and pharmacology (medical informatics) and other related fields in life science that requires computational assistance.

Bioclipse has an extensible plugin architecture and it is very easy to extend it with functionality into virtually any direction. The Bioclispe 2.0 release is targeted towards chemoinformatics, and is not to be considered stable. The core builds on a scriptable framework, and most functionality can be automated using Javascripts.

Bioclipse is the ideal framework and platform to develop and deliver software in life science to end users. Written in Java, the platform itself also supports extensions with external programs and code written in other languages.

License

Bioclipse is made up of plugins, whose code uses a number of different licenses. The Bioclipse platform, consisting of the core plugins, is licensed as EPL. This flexibel open source license allows Bioclipse plugins to have other licenses, including closed source license. Link to EPL license.
