Installing new functionality

It is easy to download and install additional features to Bioclipse. Open the menu Help > Sofware Updates and Bioclipse will contact one or more Update Sites, and let you choose additional features to istall. Note that the contents on the Update Dialog can be expanded by clicking the plus (+) sign to the left of the icon.

A number of Update Sites are preinstalled in Bioclipse download, offering a choice of features. It is easy for anyone to produce their own Update Site for Bioclipse, and you can easily add more sites to Bioclipse in the Bioclipse Preferences under the section Update Sites.

On startup, Bioclipse automatically searches for software updates if you have an Internet connection. This can be turned off in the dialog that pops up.
