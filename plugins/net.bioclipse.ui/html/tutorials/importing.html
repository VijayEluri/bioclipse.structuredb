Importing data

As mentioned in navigator help, only files inside a project in the workspace are visible in the navigator. Often, you would like to use existing data in Bioclipse. In this case, you need to import them. First create an empty project in case you do not yet have a project of which your files should become part of. Then choose "File -> Import..." or right click on the navigator (not on a file or folder) and choose "Import ...". In the wizard, choose "General -> File System" and click Next. You can then select files to import and the project to import to. Press Finish and the files will be in the project.

By choosing "General -> Existing Project into Workspace" in the wizard you can also important an existing Bioclipse project in your workspace. The project can either be already in your workspace as a directory, but not yet visible, or exist somewhere outside your workspace. This is usefull if you maintain multiple Bioclipse workspaces in parallel (which is possible) or you want to exchange complete projects with others, e. g. co-workers.
