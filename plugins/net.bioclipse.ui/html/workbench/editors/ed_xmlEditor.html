The XML-Lite Editor

The XML-Lite Editor is a basic XML-Lite Editor with syntax highlighting for editing XML-files. It is used for files with extensions as listed below, and for these files it can always be opened by right-clicking a resource in the BioResource Navigator and selecting "Open With... -> XMLEditor". The XML-Lite Editor is, as all Editors in Bioclipse, hooked up to global actions such as Copy, Paste, Undo, Redo, Select all, Find etc.

File extensions

The XML-Lite Editor is available for the following file extensions:
  • .xml
  • .cml
  • .html
  • .smr

Menu additions

There are no menu additions for the XML-Lite Editor.

Content assistance (completion)

There is no content assistance in the XML-Lite Editor.

Syntax highlighting

The XML-Lite Editor contains basic XML syntax highlighting where tags are colored in blue, and strings within tags are colored in green. Standard text between tags is displayed in black.
