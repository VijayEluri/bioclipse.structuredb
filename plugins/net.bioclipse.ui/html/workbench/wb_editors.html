Editors in Bioclipse

An Editor is focused on letting the user edit an underlying model and follows the load-save cycle. An example is a simple editor for text-files, but could also be more advanced such as a graphical objects editor like the JchemPaint editor.

Common Editor actions

Right-clicking a Views title bar gives you options to maximize, minimize, close, move etc. Doubleclicking the View's title bar is a shortcut to maximize or restore the View. Read more in Customize Bioclipse.
Most Editors in Bioclipse are hooked up to global actions such as Copy, Paste, Undo, Redo, Select all, Find etc. These are available from the main menu, section "Edit", or via right click inside the editor.

How to open an Editor

There are several ways of doing this:
  1. In the Navigator, double-click any Resource that you'd like to edit. The previously used editor will be chosen.
  2. In the Navigator, right-click a Resource and choose option: "Open". This has the same behaviour as double-clicking it.
  3. In the Navigator, right-click a Resource and choose option: "Open with..". and select the type of editor you'd like to use.
