The Console View

The Console View echoes messages to the user. This could for example be the answer to a calculation, or error messages from e.g. an executed script.

Menu additions

There are no menu additions for the Console View.

local Toolbar


Actions from left to right:
  • Clear console - clears the current console
  • Scroll lock - locks the console from scrolling
  • Pin Console - Pins the current console to remain on top of any other consoles.
  • Display selected console - realizes a listing of current consoles and allows you to select which one you would like to see.
  • Open console - allows for opening of extra consoles if installed
