Guide to Scripting Bioclipse

This is an introduction for how to use the scripting capabilities of Bioclipse.

Scripting in Bioclipse is based on Javascript. Bioclipse provides Managers that add custom funtionality. For example, the CDK Manager contributes functionality for working with cheminformatics using the Chemistry Development Kit (CDK).

Managers are access like this:

<manager name>.<manager method>(Parameters)

For example: cdk.fromSMILES("CCCO")

For a complete listing of available manager and methods, see Manager documentation.
