SMARTS Matching View

The SMARTS Matching view allows for adding/removing/editing and executing SMARTS against a chemical structure.

Toolbar and context menu actions

  • RUN matching - Execute a matching of all SMARTS. Requires an open chemical structure to operate on.
  • Clear hits - clear any hits from a previous match.
  • Expand all - Reveal all hits
  • Collapse all - Hide all hits
  • Add SMARTS - Add a new SMARTS to the View
  • Remove SMARTS - Remove an existing SMARTS from the View

Edit existing views in the Properties View (Shortcut: Context menu > Show Properties View ).
