Jmol Editor

The Jmol Editor is used to visualize molecules in 3D. It requires that the molecule has 3D coordinates to open. When opened, a new top menu is available named Jmol.


The Jmol editor: Jmol can be used for visualizing surfaces of molecules.

Click and hold the left mouse button in the view to rotate the protein. Use the mouse wheel to zoom in and out, or hold down SHIFT and click + drag at the same time. To pan around hold down SHIFT and doubleklick and drag.

Clicking atoms toggles selection on and off.

Jmol menu

The Jmol menu is available when the Jmol editor is open and it contains commands for changing the visalization in Jmol

Select

The select menu contains commands for selecting different parts of the visualized molecule.

Background

A couple of different background colors to choose from.

Atoms

This menu contains actions working on atoms. Spacefill in % sets how much big the atoms shoud be drawn.

Bonds

This menu contains actions working on bonds. Size changes the thickness of the bonds.

Surface

Surface dots can be turned on and off here.

Frame

These actions have effect on files that contains muliple frames

Biopolymer

The actions in this menu are usefull when working with for example proteins.

Labels

Actions that controls drawing of atom labels

Rotate

Actions that controls spin of the entire molecule

Export as Image

Opens a dialog from where the current Jmol visalization can be saved to a png file.
