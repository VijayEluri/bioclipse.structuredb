Jmol view

The JmolView is used to visualize molecules in 3D. It is available from the menu “Window -> Show view -> Jmol View”. If no 3D-coordinates are available, the JmolView will be empty or display “no 3D-coordinates”. When opened, a new global menu is available named Jmol.

As an example, try opening a PDB-file, for example 1D66.pdb, available from the SampleData folder that you can install using the menu alternative “install sample data”. The Jmol view displays the protein in 3D.



Click and hold the left mouse button in the view to rotate the protein. Use the mouse wheel or ???? to zoom in and out.
