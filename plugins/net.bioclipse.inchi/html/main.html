The InChI Plugin

The InChI plugin allows Bioclipse to calculate the InChI and InChIKey for molecular structures. It uses the Chemistry Development Kit and the JNI InChI library developed by Sam Adams (license:LGPL).

Platform Support

The library makes use of an JNI interface to the original InChI libraries, and such requires the latter to be available for your platform. Currently, it supports Linux (32 and 64 bit) and Mac.

Ubuntu Jaunty

Ubuntu Jaunty (9.04) users must install the libstdc++5 package.
