CML Plugin

The CML plugin mainly works in background, providing reading and writing of Chemical Markup Language (CML) files. For further information about CML, see the CML home page.

If you right click on a CML file in the BioResource Navigator, you can start a CMLValidateAction. This will tell you if the file complies with cml rules.
