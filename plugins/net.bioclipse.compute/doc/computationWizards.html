Computation Wizards

Identifier:
net.bioclipse.compute.computationWizards

Since:
0.1.0

Description:
This extension point is used to register a computation wizard. Providers may provide an extension for this extension point, and an implementation of net.bioclipse.compute.wizards.IComputationWizard.

Configuration Markup:

<!ELEMENT extension (category* , wizard* , primaryWizard*)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT wizard (selection*)>

<!ATTLIST wizard

id               CDATA #REQUIRED

name             CDATA #REQUIRED

class            CDATA #REQUIRED

icon             CDATA #IMPLIED

category         CDATA #IMPLIED

helpHref         CDATA #IMPLIED

description      CDATA #IMPLIED

descriptionImage CDATA #IMPLIED

canFinishEarly   (true | false)

hasPages         (true | false) >

  • id - a unique identifier of the wizard.
  • name - a translatable name of the wizard.
  • class - a fully qualified name of the class that implements net.bioclipse.compute.wizards.IComputationWizard interface.
  • icon - a relative path of an icon that will be used to visually represent the wizard.
  • category - the identefier of the category this wizard belongs to.
  • helpHref - the path relative to the root of the help of the document that describes this wizard.
  • description - a translatable description of the wizard.
  • descriptionImage - a relative path of an image that will be used to visually represent the wizard in the selection dialog.
  • canFinishEarly - true if this wizard can finish before all pages have been shown or false otherwise.
  • hasPages - true if this wizard needs to show pages or false otherwise.

<!ELEMENT category EMPTY>

<!ATTLIST category

id             CDATA #REQUIRED

name           CDATA #REQUIRED

parentCategory CDATA #IMPLIED>

  • id - a unique identifier of the category.
  • name - a translatable name of the wizard.
  • parentCategory - the identefier of the parent category this wizard belongs to or empty if this category is a root category.

<!ELEMENT selection EMPTY>

<!ATTLIST selection

class CDATA #REQUIRED

name  CDATA #IMPLIED>

  • class - a fully qualified name of the class for which the parent wizard must be activated.
  • name - not used.

<!ELEMENT primaryWizard EMPTY>

<!ATTLIST primaryWizard

id CDATA #REQUIRED>

  • id - the id of the wizard that must be marked as active.

Examples:
Find below an example extension of this point. See the bc_compute_demo plugin for a more detailed examples. 
 
<extension point=
"net.bioclipse.compute.computationWizards"
>

  
<category
   id=
"net.bioclipse.bc_computations.demo_category"

   name=
"%demoWizardCategory.name"
/>

  
<wizard
   category=
"net.bioclipse.bc_computations.demo_category"

   class=
"net.bioclipse.compute.demo.wizards.SimpleComputationWizard"

   description=
"%simpleComputationWizard.description"

   icon=
"icons/sample.gif"

   id=
"bc_compute_demo.wizard1"

   name=
"%simpleComputationWizard.name"
/>

  
<primaryWizard id=
"bc_compute_demo.wizard4"
/>

 
</extension>
